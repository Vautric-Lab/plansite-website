Politique de Confidentialité (Privacy Policy) Dernière mise à jour : 4 mai 2026 Bienvenue sur PlanSite. La protection de vos données personnelles est une priorité pour nous. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos informations lorsque vous utilisez notre application mobile. 1. Collecte des Données Nous collectons les informations suivantes pour assurer le bon fonctionnement de l'application : Informations de compte : Email et nom (via l'authentification Supabase). Données professionnelles : Devis, factures et informations clients que vous créez (stockés localement via Drift et synchronisés sur Supabase). Données techniques : Informations de base sur l'appareil pour garantir la compatibilité de l'application. 2. Utilisation des Données Vos données sont utilisées exclusivement pour : Gérer votre compte utilisateur et sécuriser vos accès. Synchroniser vos documents entre vos différents appareils. Améliorer nos services via l'assistant intelligent (Google Gemini API). Note : Vos données personnelles ne sont jamais utilisées pour entraîner des modèles d'IA tiers. 3. Stockage et Sécurité Stockage local : Vos données sont stockées sur votre appareil via une base de données sécurisée (Drift/SQLite). Cloud : Les données synchronisées sont hébergées sur des serveurs sécurisés via Supabase. Sécurité : Nous utilisons des protocoles de sécurité standards (RLS - Row Level Security) pour garantir que vous seul pouvez accéder à vos données. 4. Services Tiers PlanSite utilise les services suivants pour fonctionner : Supabase : Gestion de l'authentification et de la base de données cloud. Google Gemini : Traitement du langage naturel pour la recherche intelligente (sans stockage permanent de vos entrées vocales ou textuelles par Google). 5. Vos Droits Conformément au RGPD, vous disposez d'un droit d'accès, de rectification et de suppression de vos données. Vous pouvez supprimer votre compte et toutes les données associées directement depuis les paramètres de l'application. 6. Contact Pour toute question concernant cette politique, vous pouvez nous contacter à : contact@plansite.fr
