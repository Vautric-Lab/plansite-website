Politique de Confidentialité

Dernière mise à jour : 4 mai 2026

1. Collecte des Données

Nous collectons les informations suivantes pour assurer le bon fonctionnement de l'application :

  • Informations de compte : Email et nom (via l'authentification Supabase).
  • Données professionnelles : Devis, factures et informations clients que vous créez (stockés localement via Drift et synchronisés sur Supabase).
  • Données techniques : Informations de base sur l'appareil pour garantir la compatibilité de l'application.

2. Utilisation des Données

Vos données sont utilisées exclusivement pour :

  • Gérer votre compte utilisateur et sécuriser vos accès.
  • Synchroniser vos documents entre vos différents appareils.
  • Améliorer nos services via l'assistant intelligent (Google Gemini API). Note : Vos données personnelles ne sont jamais utilisées pour entraîner des modèles d'IA tiers.

3. Stockage et Sécurité

  • Stockage local : Vos données sont stockées sur votre appareil via une base de données sécurisée (Drift/SQLite).
  • Cloud : Les données synchronisées sont hébergées sur des serveurs sécurisés via Supabase.
  • Sécurité : Nous utilisons des protocoles de sécurité standards pour protéger vos informations.

4. Vos droits (RGPD)

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez d'un droit d'accès, de rectification, d'effacement et de portabilité de vos données. Pour exercer ces droits, veuillez nous contacter à l'adresse suivante : contact@plansite.fr.
